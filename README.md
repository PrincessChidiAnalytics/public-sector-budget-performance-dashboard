# 📊 Public Sector Budget & Project Performance Dashboard
## 📌 Project Overview

This project presents an interactive Excel dashboard designed to monitor public sector budget allocation, project implementation, performance, and risk across government departments. It demonstrates how data analysis can support planning, budgeting, and strategic decision-making.

## 🎯 Business Problem

Government organizations manage multiple projects with limited resources. Decision-makers need a clear way to monitor budget utilization, identify high-risk projects, and evaluate departmental performance.

This dashboard was built to provide executive-level insights that support better planning and resource allocation.

## 🛠️ Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- KPI Cards
- Dashboard Design
- Data Cleaning
- Data Analysis

## 📈 Dashboard Features
- Total Budget Overview
- Total Projects
- High-Risk Projects
- Delayed Projects
- Budget Allocation by Department
- Project Status Analysis
- Budget Category Analysis
- Interactive Department, Status, and Budget Category Filters
- Executive Insights and Recommendations

## 💡 Key Insights
- Budget allocation varies significantly across departments.
- Higher budget allocation does not always correspond to better project performance.
- Several departments have a relatively high proportion of delayed or high-risk projects.
- Interactive filters enable stakeholders to explore departmental performance and project status.
